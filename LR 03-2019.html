Проектний практикум. Лабораторна робота № 3

Клонування репозиторію засобами Git for Windows

Мета роботи

Отримати навички створення локальної робочої копії репозиторію за допомогою Git for Windows

Хід роботи

  1. Перевірити глобальні налаштування Git for Windows
  2. У випадку невідповідності вашим даним налаштувати глобальну конфігурацію Git for Windows
  3. Клонувати створений раніше репозиторій як локальну робочу копію
  4. Завантажити раніше створений репозиторій з Github в архіві. Порівняти клонований та розархівований репозиторії
  5. Для кожного етапу роботи зробити знімки екрану та додати їх у звіт з описом кожного скіншота
  6. Дати відповіді на контрольні запитання
  7. Зберегти звіт у форматі PDF
