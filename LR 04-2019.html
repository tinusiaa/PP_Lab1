Проектний практикум. Лабораторна робота № 4

Додавання файлв у репозиторій на Github

Мета роботи

Отримати навички внесення змін до локальної робочої копії та репозиторію на Github

Хід роботи

  1. Перевірити глобальні налаштування Git for Windows
  2. Клонувати створений раніше репозиторій як локальну робочу копію
  3. Додати файли до локальної робочої копії репозиторію
  4. Перевірити статус локальної робочої копії
  5. Додати до індексу локальної робочої копії нові файли
  6. Перевірити статус локальної робочої копії та у випадку відсутності непроіндексованих файлів виконати операцію фіксації (комміту)
  7. Вивантажити файли на Github
  8. Перевірити наявність нових файлів на Github через web-інтерфейс
  9. За допомогою web-інтерфейсу додати файл до репозиторію
  10. Актуалізувати локальну робочу копію та порівняти її зміст зі змістом репозиторію на Github
  11. Для кожного етапу роботи зробити знімки екрану та додати їх у звіт з описом кожного скіншота
  12. Дати відповіді на контрольні запитання
  13. Зберегти звіт у форматі PDF
